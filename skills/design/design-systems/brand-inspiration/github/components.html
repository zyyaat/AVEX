What this fixture exercises

Dense, functional, system-native.

14px body

Not 16px — GitHub's prose density is its identity. 14px base fits more content per viewport. system-ui renders instantly on every OS with zero webfont load.

Hairline borders

#d0d7de — every pane, panel, and card. Structure communicated by stroke, not shadow. Density without visual noise.

Primer blue + green

#0969da for all interactive affordances. #1a7f37 exclusively for success/merge states. Both restrained — never decorative.
